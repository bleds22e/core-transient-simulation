Average habitat values

Usage

average_habitat(locs, land)

Arguments

locs
(required) a 2-column matrix of cell coordinates
land
(required) a raster defining the landscape (see make_landscape)

Value

'A' or 'B'

Description

Calculate the habitat type that corresponds to the average habitat value across a set of cells in a landscape.
