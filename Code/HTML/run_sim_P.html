Run multiple parameter sets

Usage

run_sim_P(ncores = 1, parm_dir = "./", results_dir = "./Results/", sim_dir = NULL, report = 0, restart = F)

Arguments

ncores
number of cores to use for parallel simulation. Defaults to 1 for serial simulation.
parm_dir
directory where parameter files are located. Defaults to the current directory.
results_dir
directory where results should be saved. Defaults to 'Results' in the current directory.
sim_dir
directory where CTSim is installed, if not on the default search path
report
timestep interval at which to report simulation status to stdout. Defaults to 0 for no reporting.
restart
logical indicating whether this a restart of a previously existing set of runs. Defaults to FALSE

Value

nothing

Description

Run simulations on multiple sets of parameters

Details

This function runs simulations on parameter files stored in the directory given in parm_dir. Parameter filenames must start with 'p_'. Each parameter file should have a unique simID defined within it. Results are saved in the directory results_dir and are not returned by the function.

See also

run_simulation and restart_simulation for command line execution
