Classify occupancy vs residency

Usage

cross_classify(occupancy, breaks, b_rates = NULL, habitats = NULL, classification = NULL, do_each = F, return = "counts")

Arguments

occupancy
(required) matrix of species temporal occupancies at a set of locations (as returned by calc_occupancy)
breaks
(required) either a numeric vector of length 1 or 2 giving occupancy breakpoints for transient vs. core status or a named list with specific intervals: list(trans=c(), core=c())
b_rates
matrix of birth rates for each species in each habitat type. Dimensions are [species, habitat type].
habitats
vector of habitat types ('A' or 'B') in each location
classification
matrix classifying each species at each location as 'core' or 'trans'. Dimensions are [locations, species].
do_each
logical indicating whether one table should be returned summarizing across all locations (FALSE) or an array with one table for each spatial unit TRUE. Defaults to FALSE.
return
character string indicating whether a table of 'counts' or lists of species IDs ('ids') should be returned. Default is 'counts'.

Value

table or array, depending on do_each

Description

Cross-classify species based on their temporal occupancy and true status as core residents or transients

Details

This function classifies species as core or transient at a set of locations based on temporal occupancy and also as as biologically core or transient based on birth rates, b_rates, or a pre-defined classification. It then counts the number of species occuring in each of the four categories: biologically core - occupancy core, biologically core - occupancy transient, biologically transient - occupancy core, and biologically transient - occupancy transient. Occupancy-based core/transient categories are determined by breakpoints given in breaks. This can be either a named list with explicit intervals or a vector of one or two numbers on the interval [0,1]. In this case, transient species are those with occupancies less than or equal to breaks[1] while core species are those with occupancies greater than max(breaks). The status of species as biologically core or transient is specified either by a matrix classifying each species as 'core' or 'trans' at each spatial location (classification) or by supplying both b_rates and a vector of the habitat types at each location (habitats). This should be calculated for the same set of locations in occupancy using average_habitat.

The function can return either a count of the number of species in each category or a list of the species that are in each category (specify with return). It can also return a separate table for each location if do_each=TRUE.
