Generate propagules

Usage

reproduce(comm, b_rates)

Arguments

comm
(required) vector (or list) of species present (may contain zeros for empty spaces)
b_rates
(required) vector of birth rates for each species

Value

a vector of integers, one for each propagule

Description

Generates a pool of propagules produced by a community during one time step.

Details

comm is a vector of integers indicating which species are present. The order of b_rates designates which species each birth rate applies to. For example, b_rates[1] is the birth rate of species 1.
