Death

Usage

die(comm, m_rates)

Arguments

comm
(required) vector (or list) of species present (may contain zeros for empty spaces)
m_rates
(required) vector of mortality rates for each species in this cell (may be habitat specific)

Value

a vector of integers of the same lenth as code where 0s replace individuals that have died

Description

Determines which individuals in a community die during one time step.

Details

comm is a vector (or list) of integers indicating which species are present. Individuals in the community die with species-specific probabilities provided in m_rates. The order of m_rates designates which species each mortality rate applies to. For example, m_rates[1] is the mortality rate of species 1.
