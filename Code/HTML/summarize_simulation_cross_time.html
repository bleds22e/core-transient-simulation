Summarize simulations through time

Arguments

run_dir
directory where simulation runs are located. Defaults to the current directory.
parm_dir
directory where parameter files are located. Defaults to the current directory.
results_dir
directory where results should be saved. Defaults to 'Summaries' in the current directory.
sim_dir
directory where CTSim is installed, if not on the default search path

Description

Script used to summarize simulations on a set of parameter files through time from the command line using R CMD BATCH.

Details

This script is found in the exec/ directory where the package is installed. To run, move script to working directory and use:

R CMD BATCH '--args ncores run_dir parm_dir results_dir sim_dir' summarize_simulation_cross_time.R outfile.Rout

See also

summarize_sim_P for a description of parameters that can be passed through args
