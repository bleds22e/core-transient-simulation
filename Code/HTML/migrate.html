Immigration

Usage

migrate(m, gsad)

Arguments

m
(required) probability that an individual will arrive from outside the landscape.
gsad
(required) global relative abundance of each species. Determines probability that migrant will belong to a given species.

Value

integer representing the species that migrates. 0 = no migrant.

Description

Determine whether a species migrates into the metacommunity from outside the landscape.

Details

Species identity is determined probabilistically according to a global species abundance distribution (gsad).
