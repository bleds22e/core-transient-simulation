Aggregate habitat types

Usage

aggregate_hab_type(land, locs)

Arguments

land
(required) a raster landscape (as returned by make_landscape.
locs
(required) either a two-column matrix of cell locations defining a single spatial unit or a list of matrices designating multiple spatial units

Value

a vector of habitat types corresponding to the spatial units

Description

Determine the dominant habitat type in a set of spatial units

Details

The function determined which habitat type is present in the majority of cells in each spatial unit specified in the list locs.
