Determine species type

Usage

get_sptype(species)

Arguments

species
(required) array of species vital rates

Value

a vector of characters indicating which habitat each species prefers: 'A', 'B', or 'AB' (for a generalist).

Description

Determines whether each species in a species pool is a specialist on habitat A or B, or a generalist using birth rates.

Details

User should supply an array species in which the third dimension has a column named 'b' that stores birth rates (such as returned by make_species.)
