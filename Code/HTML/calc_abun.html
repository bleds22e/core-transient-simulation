Calculate global relative abundance

Usage

calc_abun(metacomm, N_S, only_species = F, ranked = F)

Arguments

metacomm
(required) matrix of lists defining metacommunity (as generated by populate_landscape or returned by this function).
N_S
(required) number of species expected in the metacommunity
only_species
logical indicating whether the function should ignore empty spaces. Defaults to FALSE
ranked
logical indicating whether abundances should be returned in order of rank (TRUE) or in their original order (FALSE). Default is FALSE.

Value

a vector of length N_S or, if only_species=FALSE, a vector of length N_S+1 where the first element corresponds to the number of empty spaces

Description

Calculates the relative abundance or occupancy of each species in a metacommunity

Details

The function counts the number of individuals of each species occuring in a metacommunity and then divides by the total number of individuals. If only_species is codeFALSE, then empty spaces are included in the tally and the function returns the proportion of potential spaces that each species occupies. Ranked abundances can be returned using the parameter ranked.
