Sample simulation with imperfect detection

Usage

sample_sim(abuns, probs = NULL, return = "abundance")

Arguments

abuns
(required) array of abundance profiles (as returned by calc_abun_profile)
probs
probability of observing an individual. Can be either a single detection probability for all species, or a vector with different probabilies for each species. Defaults to 1.
return
a character string indicating whether species abundances ('abundance') or presence ('presence') should be returned. Default is abundance.

Value

an array with the same dimensions as abuns

Description

Stochastically samples a simulation using a fixed detection probability.

Details

The function samples the abundances of species at a set of locations and timepoints (given in abuns) using detection probabilities in probs. These are the probabilities that an individual of a given species will be observed and can be set to be the same for all species by only providing a single number. It returns an array with the same dimensions as abun containing either the observed abundances or presence/absence (depending on return).
