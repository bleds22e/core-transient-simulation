Write a parameter file

Usage

write_parms(parm_list, f)

Arguments

parm_list
(required) list of parameters, as generated by make_parmlist
f
(required) file path

Value

nothing

export

Description

Writes a list of parameters to a file

Details

This function creates a parameter file that can be read by run_sim_P. It should be used in conjection with make_parmlist.
