Calculate dispersal vector

Usage

get_dispersal_vec(d, form = list(type = "gaussian"), N = 1)

Arguments

d
(required) numeric controlling dispersal distance (see details).
form
list defining the dispersal kernel (see details). Minimally contains an argument type defining the form of the dispersal kernel. Defaults to Gaussian.
N
number of vectors to generate

Value

a matrix of dispersal vectors. First column gives the magnitude and the second column gives the direction in radians, counter-clockwise from East.

Description

Stochastically generates dispersal vectors given a dispersal kernel and expected dispersal distance.

Details

Only implements isotropic dispersal (no directional bias). Three types of dispersal kernels are currently implemented and should be specified with the type argument in the list which is passed as the parameter form. These are:

'gaussian'
Gaussian kernel- dispersal distance sampled from a half-normal distribution with mean d.

'adjacent'
Adjacent cell dispersal (rook's move)- d is the probability that an individual will move. An additional argument moves must be passed in form giving the number of steps an individual should attempt to take.

'uniform'
Uniform dispersal- individual has equal probability of occurring anywhere within a circle with radius d. Sufficiently large d allows for unlimited dispersal.

See also

disperse for dispersal of propagules with different expected dispersal distances.
